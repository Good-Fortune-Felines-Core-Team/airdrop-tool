## Installation

Use your favourite package manager to install the CLI:

```bash npm2yarn
npm install -g @jumpdefi/airdrop-tool
```

:::note

Although optional, it is highly recommended to install the [Near CLI][near-cli] and makes sure the account you specify in the [`--accountId`](api-reference/api-reference/cli-options#--accountidaccount) is present in your credentials.

:::

## Running from command line

Once you have installed the airdrop tool globally, you can run it from the command line using:

```shell
airdrop-tool \
  --accountId="jumpfinance.near" \
  --accounts="./accounts.json" \
  --amount=100 \
  --token="jumptoken.jumpfinance.near"
```

:::note

The above example demonstrates the required options. However, see [additional options][additional-options] for more information.

:::
