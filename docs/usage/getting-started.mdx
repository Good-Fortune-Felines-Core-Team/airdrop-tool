## Installation

You can use the airdrop tool without installing it globally by using `npx`:

```bash
npx @jumpdefi/airdrop-tool --help
```

Alternatively, you can install it using your favorite package manager:

```bash npm2yarn
npm install @jumpdefi/airdrop-tool
```

:::note

Although optional, it is highly recommended to install the [Near CLI][near-cli] and make sure the account you specify in the [`--accountId`](api-reference/api-reference/cli-options#--accountidaccount) options is present in your credentials.

:::

## Running from command line

You can run the airdrop tool directly using `npx`:

```shell
npx @jumpdefi/airdrop-tool \
  --accountId="jumpfinance.near" \
  --accounts="./accounts.json" \
  --amount=100 \
  --token="jumptoken.jumpfinance.near"
```

Or if you've installed it locally:

```shell
airdrop-tool \
  --accountId="jumpfinance.near" \
  --accounts="./accounts.json" \
  --amount=100 \
  --token="jumptoken.jumpfinance.near"
```

:::note

The above example demonstrates the required options. See [CLI options][cli-options] for more additional options.

:::
